Github Commands -->
git status -- to check status of the repository
git init -- to initialize a repository
.gitignore file -- we write the name of files that we don't want to be tracked in the repository
git add . -- to add all the files in the working area
git rm --cached -- to remove the tracked files from the staging area
git commit -m "message" -- to commit all the files

git branch: Displays a list of all local branches in the repository, with an asterisk indicating the currently active branch.
git branch -a: Shows both local and remote-tracking branches. 
git branch -r: Displays only remote-tracking branches.
git branch -v: Shows local branches along with their last commit hash and commit message.

git branch <branchname> -- to create a new local branch
git branch -M <branchname> -- to create a new local branch and move to this branch

git  branch -m oldbranchname newbranchname -- to rename local branches
git branch -d branchname -- to delete a local existing branch

git remote add origin <url> -- to connect local repository to the remote repository
git push --set-upstream origin main -- to push for the first time later (git push)

git config --global user.name "username"
git config --global user.email "email"
